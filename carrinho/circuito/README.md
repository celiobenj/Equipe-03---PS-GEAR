# Circuito eletrônico do carrinho

Aqui está presente o esquema do circuito elétrico do carrinho. 

Componentes utilizados:

 * 1 Arduino UNO R3 
 * 2 motores DC 3-6V
 * 3 HC-SR04 (Ultrassônco)
 * 1 L298N (ponte H)
 * 1 bateria 9V 

Feito utilizando o software [Fritzing](https://fritzing.org/).
